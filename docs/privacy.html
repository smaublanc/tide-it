Politique de confidentialité

La version courte : Tide It ne collecte rien sur vous. La version complète est en dessous.

Dernière mise à jour : 8 juillet 2026

0 compteAucune inscription, aucun identifiant.
0 pub, 0 trackingAucun SDK publicitaire ou analytique.
Position localeTraitée sur l'appareil, jamais stockée ni partagée.
Vos données chez vousAppareil + votre iCloud personnel pour la synchro.

1.Ce que nous ne collectons pas

Tide It fonctionne sans compte utilisateur. Nous ne collectons, ne stockons et ne vendons aucune donnée personnelle : pas de nom, pas d'e-mail, pas d'identifiant publicitaire, pas de profil d'usage.

2.Localisation

Si vous l'autorisez, votre position sert uniquement, sur l'appareil, à :

  • trouver les ports, spots et balises les plus proches ;
  • afficher la météo et les conditions marines locales ;
  • créer un spot personnalisé à votre emplacement si vous le demandez.

Elle n'est jamais stockée ni transmise à des fins d'identification. Vous pouvez refuser l'accès et choisir un port manuellement — tout fonctionne.

3.Données stockées sur votre appareil

  • vos favoris, spots personnalisés et préférences d'affichage ;
  • la configuration de vos sports et alertes ;
  • le cache des marées et conditions (pour l'accès hors-ligne).

La synchronisation entre vos appareils passe par votre iCloud personnel (clé-valeur), auquel nous n'avons pas accès.

4.Services et sources de données

Tide It s'appuie sur les services Apple suivants : WeatherKit (météo), StoreKit (abonnements), MapKit (carte), iCloud (synchro de vos réglages).

Pour afficher le vent et les conditions marines, l'application interroge des sources publiques en lecture seule. Seules les coordonnées du spot consulté (ou votre position si vous l'autorisez) leur sont transmises, afin de récupérer les mesures locales. Aucune donnée personnelle, identifiant ou compte n'est envoyé, et aucune de ces sources ne dépose de traceur :

  • Open-Meteo : prévisions de vent et de mer ;
  • Pioupiou / OpenWindMap : anémomètres communautaires (licence CC-BY) ;
  • winds.mobi : réseau de balises agrégées (Holfuy, FFVL, Romma…) ;
  • aviationweather.gov (NOAA) : observations METAR des aéroports ;
  • NDBC (NOAA) : bouées météo-océaniques ;
  • Weameter : stations météo côtières.

5.Tracking et publicité

Aucun outil de tracking, d'analytique tierce, de mesure d'audience ou de publicité n'est intégré. L'application ne contient aucune publicité. Ce site web n'utilise ni cookies ni scripts de mesure.

6.Calendrier

L'accès au calendrier n'est demandé que si vous choisissez d'y exporter des horaires de marées (accès en écriture seule). Aucun événement n'est lu.

7.Notifications

Les notifications servent exclusivement aux alertes que vous configurez (marées, fenêtres GO, vent qui s'établit). Aucune notification promotionnelle.

8.Sécurité

Vos données restent sur votre appareil. Toutes les communications réseau sont chiffrées (TLS).

9.Vos droits

  • désactiver la localisation : Réglages iOS ▸ Confidentialité ▸ Localisation ▸ Tide It ;
  • révoquer l'accès au calendrier : Réglages iOS ▸ Confidentialité ▸ Calendriers ;
  • supprimer toutes les données : désinstaller l'application suffit.

Pour toute question relative à vos données (RGPD) : tideitapp@icloud.com.

10.Contact

Questions sur cette politique : tideitapp@icloud.com — ou via la page support.
