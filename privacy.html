Politique de Confidentialite

Derniere mise a jour : mars 2026

1. Introduction

Tide It respecte votre vie privee. Cette politique explique quelles donnees sont collectees et comment elles sont utilisees.

2. Donnees collectees

Tide It collecte uniquement votre position geographique lorsque vous l'autorisez, et ce uniquement pour :

  • Identifier les ports de marees a proximite
  • Afficher les donnees meteorologiques locales via Apple WeatherKit

Votre position n'est jamais stockee sur un serveur ni partagee avec des tiers.

3. Donnees stockees localement

Les donnees suivantes sont stockees uniquement sur votre appareil :

  • Port favori et preferences d'affichage
  • Configuration de vos alertes de marees
  • Donnees de marees en cache (pour un acces hors-ligne)

4. Services tiers

Tide It utilise les services Apple suivants :

  • Apple WeatherKit : donnees meteorologiques (temperature, vent, pression)
  • Apple StoreKit : gestion des abonnements Premium
  • Apple MapKit : affichage de la carte interactive

Aucun autre service tiers, SDK analytique ou reseau publicitaire n'est integre.

5. Tracking et publicite

Tide It n'utilise aucun outil de tracking, d'analytique, de publicite ou de collecte de donnees a des fins commerciales. L'application ne contient aucune publicite.

6. Calendrier

L'acces a votre calendrier est demande uniquement si vous choisissez d'exporter les horaires de marees. Aucun evenement n'est lu depuis votre calendrier.

7. Notifications

Les notifications push sont utilisees exclusivement pour les alertes de marees que vous configurez. Aucune notification promotionnelle n'est envoyee.

8. Securite

Toutes les donnees restent sur votre appareil. Les communications avec les services Apple sont chiffrees via TLS.

9. Droits de l'utilisateur

Vous pouvez a tout moment :

  • Desactiver la localisation dans Reglages > Tide It
  • Supprimer toutes les donnees en desinstallant l'application
  • Revoquer l'acces au calendrier dans Reglages > Confidentialite

10. Contact

Pour toute question concernant cette politique : tideit.app@gmail.com

11. Modifications

Cette politique peut etre mise a jour. La date de derniere modification est indiquee en haut de cette page. L'utilisation continue de l'application vaut acceptation des modifications.
